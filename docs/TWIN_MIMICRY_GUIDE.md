# Twin Mimicry Guide

## Overview
Twin Mimicry (`twin-mimicry.ts`) allows the system to analyze and adopt the structural and logical patterns of elite repositories.

## Configuration
- **Target Repo**: Source repository for pattern extraction.
- **Style**: Exact, Balanced, or Creative.

## Usage
Used during code generation or architectural planning tasks.

## Troubleshooting
- Ensure the target repository is public and accessible via GitHub API.
