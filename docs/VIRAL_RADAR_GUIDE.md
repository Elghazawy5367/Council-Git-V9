# Viral Radar Guide

## Overview
Viral Radar (`viral-radar.ts`) tracks emerging trends across social platforms to provide real-time context to the Council.

## Configuration
- **Platforms**: Twitter, Reddit, HackerNews.
- **Sensitivity**: Adjustable viral score threshold in Feature Settings.

## Usage
Used by the "Trend Analyzer" expert to ground suggestions in current events.

## Troubleshooting
- Check Serper API usage if trends stop updating.
