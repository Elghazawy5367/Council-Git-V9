# When to Use Google Studio vs IDX

## Use IDX AI for:
- Quick syntax fixes
- Refactoring existing code
- Debugging errors
- Simple questions

## Use Google Studio for:
- Implementing new features (>100 lines)
- Architectural decisions
- Complex algorithm design
- When IDX says "message limit reached"
